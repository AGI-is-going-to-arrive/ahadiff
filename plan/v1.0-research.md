## 8. Graphify 深度集成专项

> **官方仓库**: https://github.com/safishamsi/graphify
> **行动项**: 更新本地 graphify 到最新版，全面测试后再决定集成深度

### 当前状态
- v0.1 定位："可选增强，不是主链前置"
- `ahadiff learn` 自动检测 `graphify-out/graph.json`
- CLI: `ahadiff graph status/refresh/import`
- `_AncestryCache` 有 200 subprocess cap
- Viewer ConceptGraph 页面有基础力导向图

### v1.0 集成方案

| ID | 任务 | 优先级 | 依赖 |
|----|------|--------|------|
| GR-1 | 更新 graphify 到最新版 + 回归测试 | P0 | 无 |
| GR-2 | `_normalize_id()` fuzzy matching | P1 | GR-1 |
| GR-3 | Freshness API（7 态 → 4 值投影展示） | P1 | GR-1 |
| GR-4 | ConceptGraph typed nodes + filter chips + legend | P1 | GR-3 |
| GR-5 | Graph Graphify source card（sync 状态、CLI 命令） | P2 | GR-1 |
| GR-6 | graph.json FTS5 索引（可选） | P2 | B1.2 |
| GR-7 | PreToolUse hook 条件上下文注入 | P2 | GR-1 |
| GR-8 | `_AncestryCache` 大 repo 性能优化 | P2 | GR-1 |

### 后端深度研究结论
- **当前状态**: `detect_graphify_status()` 仅实现 7 态中的 1 态（binary source_present）；concept 和 graph node **完全未链接**
- **Fuzzy matching**: Jaccard token similarity (threshold 0.85) 在 `compute_term_key()` 之上。~80-120 行 `wiki/matcher.py`
- **Freshness**: mtime→commit-hash 比较（import 时存 HEAD 到 `graphify_meta.json`）
- **AncestryCache**: 200 subprocess cap 对 1000+ runs 不够 → 改为 `commit_ancestry` SQL 表 + bulk `git log`
- **3 Critical review 待闭合**: C1 optional-dep 耦合、C2 缺 Pydantic models、C3 mtime freshness
- **Bug**: `total_reviews` 计数器从未递增，阻塞 FSRS Optimizer

### 前端深度研究结论
- **ConceptGraph 不是 placeholder** — 已有 circular layout SVG + click 选择 + 3-mode display
- **图渲染库**: 推荐 **d3-force 模块化导入 (~15KB)**，替换 `computeCircularLayout()` 为 force simulation，保留 React JSX SVG
- react-force-graph-2d (180KB) 太重且抽象掉 SVG 控制
- **数据模型差距**: 前端 `Concept` 接口丢弃了 `parents`, `children`, `relationship_type`, `source_refs` 等字段
- **V6 需要 5 种 node 类型**: repo context / current diff / symbol / verified / weak
- **Freshness 展示**: Dashboard badge + Settings Graphify tab + ConceptsPage source card 三处
- **前端 Graphify 工作量**: ~860 LOC (640 TSX + 220 CSS)

### 决策点
- **集成深度**: v1.0 "条件深度集成"——有 graphify 则一等公民展示，无则静默降级，不自动安装（supply-chain 原则）
- **图渲染库**: d3-force 模块化导入 (~15KB)，不用 d3 全家桶
- **PreToolUse hook**: v1.0 不做（13 个 agent target 太复杂），改用 `[graphify] auto_refresh = true` config
