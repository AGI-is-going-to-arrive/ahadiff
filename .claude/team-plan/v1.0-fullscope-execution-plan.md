### 0.1 Current Backend Restart Point

后续后端继续时，不该回到 Phase 0 从头重做。当前分支更合理的起点，是先承认 **0G 已收口、3E 已接线、3D 已大体落地**，然后 backend-first 收真正还没闭合的链：

1. **承认 1B / DAO 并发决策已经有结论**：当前 serve 路径已经通过 threadpool 基准与文档裁决，继续维持 `anyio.to_thread.run_sync` + 同步 `sqlite3`，不引入 `aiosqlite`。
