现在很多人用 AI 助手写代码。Cursor、Claude Code、Copilot，一天能改几十个文件。你 commit 了，代码能跑了，但你真的知道改了什么吗？这就是 vibe coding 的知识黑洞：AI 帮你实现了功能，但这些改动从来没有变成你自己的知识。下次遇到类似问题，你还是得再问 AI。知返就是为了解决这个问题而做的。

使用知返非常简单。知返目前还没发布到 PyPI，在源码目录里用 uv tool install --editable . 安装命令行，或者直接用 uv run ahadiff 运行。安装好之后，进入你的项目目录，运行 ahadiff init 初始化。然后 AI 帮你改完代码后，运行 ahadiff learn 加上你要学习的提交范围。比如 HEAD~1..HEAD 就是学最近一次提交，加 --staged 就是学暂存区的改动，加 --unstaged 就是学工作区还没 add 的改动。你还可以用 --changed-path 指定只学某个文件。学完以后，用 ahadiff serve 打开网页界面查看结果。Guide 页面列出了所有命令，包括 quiz、review、verify、improve 等，还有进阶的数据库维护和概念管理命令。

知返不是给你生成一段 commit 摘要就完了。它的核心思想是带证据的学习：课程里每一句话都必须指向具体的文件和行号。打开 Diff 页面，你能看到这次代码改动，左侧有彩色小圆点标记课程提到的代码行。点击圆点，右侧的检查面板会显示对应的解释和代码片段。每条结论有五种验证状态：已验证、较弱、未证明、被反驳、已拒绝。这样你学到的不是模型的幻觉，而是可以追溯到代码的真实知识。

Lesson 页面展示这次改动的完整课程。知返会生成三档课程：完整版适合深入学习，提示版只保留关键点，精简版用最少的篇幅概括改动。课程顶部会显示这次学习的评分和通过状态。你可以点击「看评分细节」进入评分页面，或者点击「看 LLM 评审」查看模型的语义评审意见。这些评分和评审帮你判断这次生成的课程质量是否可靠。

光读课程很容易忘。知返用两种方式帮你记住：Quiz 页面会出选择题，你得自己想答案，这叫主动回忆，比被动阅读有效得多。做完测验后，知识点会变成复习卡片。Review 页面用间隔重复算法安排复习时间，在你快要忘记的时候提醒你复习。每张卡片按掌握程度打分，系统自动调整下次复习间隔。这里展示的是我本地开发测试时积累的数据，你安装知返后需要自己运行 learn 命令才会生成属于你的课程和卡片。

知返用 8 个维度给每次学习打分，满分 100。准确性看课程内容是否正确；证据绑定看每句话有没有对应的代码来源；差异覆盖看课程是否覆盖了这次改动的主要文件；可学习性看课程是否适合学习；测验迁移看测验题和课程内容是否对应；规范对齐看是否满足你指定的学习要求；简洁性看篇幅是否合理；安全与隐私看有没有泄露敏感信息。其中准确性和证据绑定有硬门槛，低于门槛直接判定不合格。你可以在运行详情页面的评分和评审标签页查看每个维度的具体分数和 LLM 评审意见。

知返不只是学一次就丢。每次学习都会提取新的概念，累积到概念台账里。台账页面展示所有学过的知识点、它们的来源文件和健康状态。切换到图谱视图，你能看到概念之间的关系网络，节点是知识点，边是它们的关联。这张图会随着你学习更多的代码改动不断生长。这里的数据是我本地测试积累的，你需要在自己的项目里运行 learn 才会逐渐生成属于你的概念图谱。顺便说一下，知返的概念累积设计借鉴了 Karpathy 的 LLM Wiki 思路和 Graphify 代码图谱工具，但落地为按 diff 维度叠加的学习层。

运行面板展示你所有的学习记录、总分趋势和待复习数量。你能看到哪些学习通过了，哪些没通过，分数是上升还是下降。设置页面用来配置你自己的 LLM provider，支持 OpenAI、Gemini、Anthropic 等多种格式，填上接口地址和密钥就行。你也可以用 ahadiff doctor 检查环境、用 ahadiff config show --resolved 查看当前配置。知返还支持 13 种 AI 编码工具的项目级集成，包括 Claude Code、Cursor、Copilot、Codex 等，一条 ahadiff install 命令就能写入对应工具的指引文件。所有学习记录都保存在你的项目目录里，模型调用由你自己配置和管理。

总结一下。知返解决的核心问题是：AI 帮你改了代码，但你没有从中学到东西。它用带证据的学习方式，确保你学到的是真实的、可以追溯到代码的知识。在源码目录里用 uv tool install 安装知返，运行 ahadiff init 初始化项目，配置你的模型 provider，然后对最近的一次改动运行 learn。打开网页看课程、查证据、做测验、定期复习。知返还支持 13 种 AI 编码工具的项目级集成，让你的 AI 助手也知道怎么使用知返。让 AI 负责写代码，让知返把知识还给你。
