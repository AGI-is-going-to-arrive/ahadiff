THE PROBLEM

Vibe Coding 的知识断层

AI 工具让你 10x 产出，但学习曲线变成了一条平线。

AI 写的代码看不懂

Copilot / Claude 产出大量代码，你点了 Accept，却说不清它为什么有效、何时会失败。

Code Review 成了橡皮图章

diff 太大、上下文不够，review 变成形式主义。关键逻辑在 LGTM 中悄悄滑过。

知识债务无声堆积

用 AI 交付越快，技术债中「人脑里欠的那部分」累积越深，直到一次线上事故暴露一切。

FEATURES

每个细节都为学习设计

不是又一个文档工具，而是一个让你和 AI 共同成长的学习系统。

Verified Claims

每句解释都链接到 file:line 代码证据。5 种 claim 状态 (verified / weak / not_proven / contradicted / rejected) 确保可信。

FSRS Spaced Repetition

采用 Anki 同源的 FSRS-6 算法，根据你的答题表现自动调整复习间隔。

Three-Stage Scaffolding

Full → Hint → Compact 三阶段学习撤架。由 FSRS 记忆稳定度自动驱动，越熟悉提示越少。

8-Dimension Rubric

accuracy / evidence / diff_coverage / learnability / quiz_transfer / spec_alignment / conciseness / safety_privacy = 100 分。

Git Ratchet

只进不退：improve loop 渐进优化，Phase 2.5 在卡住时触发结构性重写，突破质量瓶颈。

Privacy First

strict_local / redacted_remote / explicit_remote 三档隐私。Secret scan 在所有操作之前执行，绝不泄露。

8 LLM Providers

OpenAI / Anthropic / Gemini / Azure / Ollama 等 8 种 API 格式。BYOK 自动探测 context length、TPM/RPM。

Multi-Agent Install

Claude Code / Codex CLI / Gemini CLI / OpenCode 一键接入，AI agent 在你的工作流中无缝集成。

PLANNED INTERFACE

三步开始（计划中）

从安装到第一个学习循环，不超过两分钟。

1

Install

通过 pip 安装 AhaDiff CLI 工具。

$ pip install ahadiff
2

Learn

在你的项目中运行 learn 命令，分析最近一次提交的 diff。

$ cd your-project
$ ahadiff learn --last
3

Review

打开 Web UI 进行 SRS 复习和测验，巩固知识。

$ ahadiff serve
